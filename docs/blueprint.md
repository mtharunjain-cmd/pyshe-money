# **App Name**: PsycheMoney

## Core Features:

- User Authentication: Secure sign-in functionality to manage student financial data.
- Menu Options: Left-corner menu with navigation to key features.
- Transaction Tracker: Real-time tracker for income and expenses.
- Financial Ratios: Calculate and display key financial ratios.
- Premium Subscriptions: Offer premium subscription tiers (99, 499, 999).

## Style Guidelines:

- Primary color: Calm, professional blue (#5DADE2), evocative of trust.
- Background color: Very light blue (#EBF5FB), provides a clean, uncluttered look.
- Accent color: Muted violet (#8E44AD), offers a touch of sophistication and calls attention to premium features.
- Headline font: 'Poppins', a geometric sans-serif with a contemporary, precise feel.
- Body font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look.
- Use simple, clean icons to represent different financial categories and menu options.
- Clean, intuitive layout that's easy to navigate, even on mobile devices.
- Subtle transitions and animations to improve user experience when tracking transactions.
